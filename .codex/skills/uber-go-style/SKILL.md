# Uber Go Style

## 使い方

- `references/uber-go-style-guide.md` を読み、該当セクションのみ適用する。
- 判断が曖昧な場合は原典（uber-go/guide の `src/` と `style.md`）を優先する。
- レビュー時は「設計 → エラー → 並行性 → 構造 → 性能 → 命名/スタイル → テスト」の順で確認する。
